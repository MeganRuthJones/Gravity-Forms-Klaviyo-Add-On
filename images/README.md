# Images Directory

This directory should contain the Klaviyo menu icon SVG file.

## Required File

- `menu-icon.svg` - The SVG icon for the Klaviyo add-on menu item in Gravity Forms settings

The icon should be an SVG file with the class `gf-klaviyo-icon` on the root `<svg>` element for proper styling.

Example:
```xml
<svg class="gf-klaviyo-icon" xmlns="http://www.w3.org/2000/svg" viewBox="0 0 48 48">
  <!-- Icon content -->
</svg>
```
